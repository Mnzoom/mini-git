# mini-git
Mini-Git est un système de versionnage simplifié qui permet de créer un dépôt, effectuer des commits, consulter l’historique et restaurer des snapshots de fichiers. L’objectif est de maîtriser la programmation bas niveau et la gestion de structures complexes.


1 Me 
Architecture & init repo
Conception système, modularité, logique d’alertes

2
Commit / Checkout
Parsing efficace, manipulation de fichiers, analyse de données

3
Log / Tests unitaires
Algorithme de détection, gestion des edge cases, tests sécurité
